Game of Nin

This Python project exposed me to object oriented programming and machine learning. The bot learned by tracking its moves and adding points to those moves if it won the game. It took away points from those moves if it lost the match. The bot would prefer the move that had the most points.
  • Worked as a team to complete task well before the deadline and meet all requirements
  • Created a simple interface for user to make moves and decide who would play (user vs user, user vs bot, bot vs bot)
  • Developed rudimentary machine learning algorithm
